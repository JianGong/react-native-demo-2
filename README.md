#Same Ride demo

###Supported input

To see how map widget works, you need to enter on of theese cities:

 - New York
 - Jersey City
 - Newark
 - Irvington
 - Nutley

###Supported devices

Application was tested on iPhone 5s, 5c, 6, 6 plus, 6s, 6s plus
